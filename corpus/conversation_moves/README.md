# `corpus/conversation_moves/`

Bernard-authored examples of the six clinical moves enumerated in
[`api/insult_ai/policies/clinical_moves.md`](../../api/insult_ai/policies/clinical_moves.md):

- `validation`
- `reflection`
- `cognitive_reframe`
- `behavioral_activation`
- `planning`
- `boundary`

Each file in this directory is one `.md` document per move, showing
short worked examples of the move in the Roast Coach voice
(see `personas/clinical_compadre.md`). These are NOT scraped or
licensed-in; they are original work by Bernard. License: same as the
repo (see `LICENSE`).

This directory is **empty in Slice 1**. Files will be added in Slice 2.

Documents added here must still carry the same frontmatter spec as
`clinical_public_knowledge/` (see
[`docs/research/corpus_methodology.md`](../../docs/research/corpus_methodology.md)).
Use `source: insult_ai` and `license: project-original` (which must be
added to the ingest allowlist when Slice 2 lands).
